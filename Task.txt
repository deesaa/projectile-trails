Тестовая задача:

Реализовать широкий дымовой след от пули в виде сетки (mesh), строящейся за пулей, летящей по баллистической траектории. 
Новый сегмент сетки добавляется, когда пуля пролетает заданное расстояние (segmentLength).
Каждый сегмент сетки должен быть повернут к динамической камере. 
Задавать UV-координаты, цвета вершин для плавного рассеивания не нужно.
Обратите внимание на организацию данных в памяти, количество объектов и вызовов отрисовки.

Использовать стандартный TrailRenderer не надо, суть задачи в самостоятельной реализации подобного компонента в упрощенном виде.
Для отображения полученного меша использовать MeshRenderer.

Необходимо сделать реализацию и предложить методы оптимизации производительности при большом количестве следов.

Прикреплен проект на Unity.
Подготовлена шаблонная сцена "Scene".
Необходимо реализовать класс ProjectileTrailRenderer, в котором добавлены события на создание, удаление и движение пули (структура Gun.Projectile).
Индекс пули - идентификатор в заранее известном диапазоне от 0 до gun.maxProjectileCount-1 включительно.
Для каждого снаряда собственный индек постоянен, для новых снарядов индексы назначаются циклически.

Результат должен быть похож на скриншот.

Выполнение задачи не должно занять больше трех часов.
После выполнения задания необходимо прислать весь проект, включая сборку c исполняемым файлом, для запуска и проверки.
